DuBo Martial Arts

Proyecto realizado para el grado superior de Desarrollo de Aplicaciones Web en la asignatura Desarrollo Web en Entorno Cliente.

Descripción

Este proyecto como trabajo de la asginatura de Desarrollo Web en Entorno Cliente, consiste en la creación de una página web para un gimnasio de artes marciales. El proyecto se ha realizado con HTML, CSS y JavaScript con un servidor Express y una API con JSON.

Herramientas utilizadas

HTML
CSS
JavaScript
Bootstrap
NodeJS
Express
